PharmaBot: Autonomous Medicine Delivery

E-Yantra Competition - IIT Bombay

Overview

The PharmaBot project was developed as part of the E-Yantra Robotics Competition organized by IIT Bombay. The objective was to design a robot capable of autonomously delivering medicines from a pharmacy to designated houses using efficient path planning and intelligent control.

Key Features

  • Path Planning: Implemented Dijkstra's algorithm to compute shortest routes
  • Computer Vision: Used OpenCV for detecting road markers and navigation
  • Controller: Raspberry Pi managed sensors and motors
  • Simulation: CoppeliaSim for testing and validation

Technologies Used

Python Raspberry Pi OpenCV CoppeliaSim Dijkstra's Algorithm

Two-Wheeled Inverted Pendulum

Advanced Control Strategies

Overview

This project focused on deriving the dynamics of a two-wheeled inverted pendulum and applying advanced control techniques to stabilize it. Dynamics were modeled in MATLAB and tested in MuJoCo.

Control Strategies

  • Carleman+LQR
  • DSC+NN+NMPC
  • EPSAC
  • H_inf
  • LPV
  • LQR+L1 adaptive
  • LQR+SMC+Backstepping
  • LQR
  • MPC
  • Pole Placement
  • SMC

Technologies Used

MATLAB MuJoCo Control Systems Robot Modelling Optimization
