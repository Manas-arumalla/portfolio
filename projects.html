Mobile Robot Path Planning Algorithms

Comparative Algorithm Analysis

Overview

A comprehensive implementation of 7 path planning algorithms for differential-drive mobile robots with enhanced visualization capabilities. The project emphasizes comparative algorithm analysis with a modular architecture for easy extension.

Key Features

  • Grid-Based: A* (Euclidean heuristic) and Dijkstra (uniform cost) - both optimal
  • Sampling-Based: RRT, RRT* (asymptotic rewiring), PRM (Probabilistic Roadmap)
  • Optimization-Based: PSO (Particle Swarm Optimization) - locally optimal
  • Reactive: APF (Artificial Potential Fields)

Technical Highlights

  • Dark-theme visualization with algorithm-specific color palettes
  • Path gradient effects and glow rendering
  • Robot simulation with trajectory tracking
  • Performance metrics (computational time, path length analysis)

Technologies Used

Python NumPy Matplotlib SciPy A* RRT PRM PSO

ROS 2 + PlanSys2 Integration

6-DOF Robotic Arm Trajectory Planning

UR5-Based Simulator

Overview

A Python-based simulator for robotic arm control demonstrating forward/inverse kinematics, trajectory planning, and closed-loop feedback control. Based on the UR5 industrial robot with 6 revolute joints in anthropomorphic configuration.

Trajectory Planners

  • Cubic Polynomial: Smooth position and velocity profiles
  • Quintic Polynomial: Adds smooth acceleration constraints
  • Trapezoidal Velocity: Constant acceleration/deceleration phases

Key Features

  • Forward/Inverse Kinematics using Denavit-Hartenberg convention
  • Multi-waypoint path planning support
  • PID control with anti-windup mechanisms
  • 3D visualization comparing desired vs actual end-effector paths
  • Comprehensive performance analysis plots

Technologies Used

Python NumPy SciPy Matplotlib DH Convention PID Control Kinematics
